# Betaflow — Pipeline simple (TXT → PDF)

Pequeño pipeline llamado Betaflow que recibe un .txt como entrada y devuelve un .pdf como salida.

## Requisitos

Python 3.10+ (o contenedor Docker)

Dependencias (si usas Python): pip install -r requirements.txt

```sh
# 1) Coloca tu archivo de entrada
echo "Hola Betaflow" > input.txt

# 2) Ejecuta el pipeline
betaflow --input input.txt --output salida.pdf

# 3) El PDF quedará en:
ls -lh salida.pdf
```
